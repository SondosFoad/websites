Head Tags and Their Uses

The tags that are used in head part are:

  1. title
    2. The title tag defines the title of the page that appears in the browser's title bar and tabs.
  2. meta
    3. This tag provides metadata about the page.
    • name:
      • description: Provides a brief description of the page content, helping search engines and users to understand the content better.
      • keywords: A list of keywords related to the page content, helping to improve the page's ranking in search results.
      • author: Specifies the name of the author or person responsible for the page content.
      • viewport: Defines how the page will appear on mobile devices, such as width=device-width and initial-scale=1.0.
      • robots: Directs search engines on how to index the page (e.g., "noindex" or "nofollow").
      • theme-color: Specifies the color of the status bar in mobile browsers.
    • content: Contains the value associated with the specified property.
    • charset: Specifies the character encoding used, such as "UTF-8".
  3. link
    4. This tag is used to link CSS files or other external resources like icons. It allows you to separate styles from content.
    • rel: Specifies the type of relationship between the document and the linked resources (e.g., "stylesheet").
    • href: Specifies the URL of the linked file.
    • type: Specifies the content type, such as "text/css".
  4. style
    5. This tag allows you to write CSS styles directly within the page.
    • type: Specifies the content type, usually "text/css".
    • scoped: If used, it indicates that the styles apply only to this element.
  5. script
    6. This tag is used to add JavaScript to the page.
